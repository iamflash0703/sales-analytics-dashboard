## 🎯 Project Overview

This project demonstrates end-to-end data analysis skills by:
1. **Cleaning** a messy real-world sales dataset (5,000+ records)
2. **Analyzing** the data to extract business insights
3. **Visualizing** findings with interactive charts
4. **Building** a live web dashboard with Streamlit

## 📊 Key Insights Discovered

| Metric | Value |
|--------|-------|
| **Total Revenue** | $1,363,619.91 |
| **Total Orders** | 4,947 |
| **Unique Customers** | 797 |
| **Average Order Value** | $275.65 |

### Top Findings:
- 🏆 **Electronics** is the highest revenue category (65.3% of total)
- 🌍 **West Region** leads in sales (27.0% market share)
- 💻 **Laptop** is the best-selling product
- 💳 **Credit Card** is the most preferred payment method
- 📈 Sales peak during **November-December** (holiday season)

## 🧹 Data Cleaning Steps Performed

1. ✅ Removed **45 duplicate rows**
2. ✅ Filled **252 missing values** with appropriate defaults
3. ✅ Standardized text data (case, spaces, typos)
4. ✅ Fixed data types (Discount_Percent from text to number)
5. ✅ Parsed and validated dates (removed invalid & future dates)
6. ✅ Removed **15 rows** with negative quantities
7. ✅ Removed **8 rows** with impossible prices
8. ✅ Recalculated Total_Amount with correct formula
9. ✅ Created derived columns (Year, Month, Quarter, Day_of_Week)
